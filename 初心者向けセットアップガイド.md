## Node.jsのインストール

### ステップ1: Node.jsをダウンロード

1. ブラウザ（Edge、Chromeなど）を開きます

2. アドレスバーに以下を入力してEnterキーを押します：
   ```
   https://nodejs.org/ja
   ```

3. 緑色の大きなボタン「**推奨版**」をクリックしてダウンロード
   - 「LTS」や「推奨版」と書かれている方を選んでください
   - バージョン番号は気にしなくてOKです

4. ダウンロードが完了するまで待ちます（数分）

### ステップ2: Node.jsをインストール

1. ダウンロードしたファイル（`node-vXX.XX.X-x64.msi`）をダブルクリック

2. インストーラーが開いたら、以下の手順で進めます：
   - **「Next」ボタン**を押す
   - **「I accept...」にチェック**を入れて「Next」
   - **そのまま「Next」**を何回か押す
   - **「Install」ボタン**を押す
   - ユーザーアカウント制御が出たら**「はい」**をクリック
   - インストール完了まで待つ（数分）
   - **「Finish」ボタン**を押す

3. インストール完了！

### ステップ3: インストール確認

1. キーボードの **Windowsキー** を押して、「cmd」と入力

2. 「**コマンドプロンプト**」が表示されるのでクリック

3. 黒い画面（コマンドプロンプト）が開きます

4. 以下を入力してEnterキーを押します：
   ```
   node --version
   ```

5. `v18.XX.X`のような表示が出ればOK！
   - 数字は違っていても大丈夫です

6. コマンドプロンプトの右上の「×」で閉じます

## プロジェクトの準備

### ステップ1: プロジェクトをローカルにコピー

**重要**: Googleドライブ上では動作しないため、パソコンの中にコピーします

1. エクスプローラー（フォルダのアイコン）を開きます

2. 現在のプロジェクトフォルダを開きます：
   ```
   G:\マイドライブ\Cursor\ChatBot制作
   ```

3. このフォルダ全体を**コピー**します：
   - フォルダを右クリック → 「コピー」

4. コピー先に移動します（どちらか選んでください）：
   
   **おすすめ: ドキュメントフォルダ**
   - 左側のメニューから「ドキュメント」をクリック
   - 空いているところで右クリック → 「貼り付け」
   
   **または: Cドライブ直下**
   - 「PC」→「ローカルディスク(C:)」を開く
   - 空いているところで右クリック → 「貼り付け」

5. コピーが完了するまで待ちます（数秒〜1分）

### ステップ2: コピーした場所を確認

コピーした場所は、例えばこのようになります：

**ドキュメントフォルダの場合**:
```
C:\Users\[あなたのユーザー名]\Documents\ChatBot制作
```

**Cドライブ直下の場合**:
```
C:\ChatBot制作
```

このパス（場所）を**メモ帳などに控えておいて**ください！

### ステップ3: コマンドプロンプトを開く

1. キーボードの **Windowsキー** を押して、「cmd」と入力

2. 「**コマンドプロンプト**」をクリック

3. 黒い画面が開きます

### ステップ4: プロジェクトフォルダに移動

1. コマンドプロンプトに以下を入力します：

   **ドキュメントにコピーした場合**:
   ```
   cd Documents\ChatBot制作
   ```
   
   **Cドライブにコピーした場合**:
   ```
   cd C:\ChatBot制作
   ```

2. Enterキーを押します

3. 画面の表示が変わればOK：
   ```
   C:\Users\...\ChatBot制作>
   ```
   または
   ```
   C:\ChatBot制作>
   ```

### ステップ5: 必要なファイルをインストール

1. 以下を入力してEnterキーを押します：
   ```
   npm install
   ```

2. 画面に色々な文字が流れます（**これは正常です**）

3. 30秒〜2分待ちます

4. 最後に以下のような表示になればOK：
   ```
   added XXX packages in XXs
   ```

5. エラーが出た場合は、[よくあるエラーと解決方法](#よくあるエラーと解決方法)を参照してください

## サーバーの起動

### ステップ1: サーバーを起動する

1. コマンドプロンプトで以下を入力してEnterキーを押します：
   ```
   npm start
   ```

2. 数秒待つと、以下のような表示が出ます：
   ```
   サーバーが起動しました: http://localhost:3000
   チャットボット: http://localhost:3000/chatbot.html
   FAQ管理画面: http://localhost:3000/admin.html
   ```

3. **この画面は閉じないでください！**
   - サーバーが動いている証拠です
   - 閉じるとチャットボットが使えなくなります

### ステップ2: ブラウザでアクセス

1. ブラウザ（Edge、Chromeなど）を新しく開きます

2. アドレスバーに以下を入力してEnterキー：
   ```
   http://localhost:3000/chatbot.html
   ```

3. チャットボット画面が表示されればOK！🎉

## チャットボットの使い方

### チャットボット画面の使い方

1. ブラウザで以下にアクセス：
   ```
   http://localhost:3000/chatbot.html
   ```

2. 画面の説明：
   - **上部**: チャットボットのヘッダー（紫色）
   - **中央**: 会話が表示されるエリア
   - **下部**: よくある質問のボタン
   - **最下部**: メッセージ入力欄

3. 使い方：
   - 「よくある質問」ボタンをクリックするだけでOK
   - または、入力欄に質問を入力して紙飛行機ボタンをクリック

4. チャットボットが自動で返答します！

### FAQ管理画面の使い方

1. ブラウザで以下にアクセス：
   ```
   http://localhost:3000/admin.html
   ```

2. 左側：新しいFAQを追加
   - **質問**: ユーザーが聞く質問を入力
   - **回答**: チャットボットの返答を入力
   - **キーワード**: 検索用のキーワード（カンマで区切る）
   - 「追加」ボタンをクリック

3. 右側：登録済みFAQの一覧
   - **✏️ 編集**: FAQを修正できます
   - **🗑️ 削除**: FAQを削除できます

### キーワードの設定例

**質問**: 営業時間を教えてください

**キーワード**: 
```
営業時間, 何時, いつ, 時間, 営業, オープン, 開店
```

こうすることで、ユーザーが「何時まで営業してますか？」と聞いても正しく答えられます。

## よくあるエラーと解決方法

### エラー1: 「npm: コマンドが見つかりません」

**原因**: Node.jsがインストールされていない

**解決方法**:
1. [Node.jsのインストール](#nodejsのインストール)を最初から実行
2. インストール後、コマンドプロンプトを**一度閉じて開き直す**

### エラー2: 「ポート3000は既に使用されています」

**原因**: 別のプログラムがポート3000を使っている

**解決方法A（簡単）**:
1. Ctrl + C キーを押してサーバーを停止
2. もう一度 `npm start` を実行

**解決方法B（ポート番号を変更）**:
1. `server.js` を開く（メモ帳でOK）
2. 3行目あたりの以下を探す：
   ```javascript
   const PORT = process.env.PORT || 3000;
   ```
3. `3000` を `3001` に変更：
   ```javascript
   const PORT = process.env.PORT || 3001;
   ```
4. 上書き保存
5. `npm start` を再実行
6. ブラウザで `http://localhost:3001/chatbot.html` にアクセス

### エラー3: 「Cannot find module 'express'」

**原因**: `npm install` が正しく完了していない

**解決方法**:
1. コマンドプロンプトで以下を実行：
   ```
   npm install express cors body-parser
   ```
2. 完了後、`npm start` を再実行

### エラー4: チャットボットが応答しない

**原因**: サーバーが起動していない、またはURLが間違っている

**解決方法**:
1. コマンドプロンプトで「サーバーが起動しました」と表示されているか確認
2. 表示されていない場合は `npm start` を実行
3. ブラウザのURLが正しいか確認：
   ```
   http://localhost:3000/chatbot.html
   ```
4. ブラウザでF12キーを押して、コンソールにエラーが出ていないか確認

### エラー5: 「npm install」で大量のエラー

**原因**: Googleドライブ上で実行している

**解決方法**:
1. [プロジェクトの準備](#プロジェクトの準備)のステップ1を確認
2. 必ずパソコンのローカルディスク（CドライブやDocuments）にコピー
3. コピーした場所で `npm install` を実行

## 💡 豆知識

### サーバーを停止する方法

コマンドプロンプトで **Ctrl + C** キーを押します。
「バッチ ジョブを終了しますか (Y/N)?」と出たら **Y** を入力してEnter。

### サーバーを再起動する方法

1. Ctrl + C でサーバーを停止
2. `npm start` でサーバーを起動

### ファイルを編集した後

FAQデータ（`faq-data.json`）を編集した場合は、ブラウザを再読み込み（F5キー）するだけでOK。

プログラムファイル（`server.js`など）を編集した場合は、サーバーを再起動する必要があります。
