│  よくあるエラーと解決方法                                           │
└─────────────────────────────────────────────────────────────────────┘

【エラー】
┌────────────────────────────────────┐
│ 'npm' は、内部コマンドまたは外部   │
│ コマンド...として認識されていません│
└────────────────────────────────────┘
【原因】Node.jsがインストールされていない
【対処】Node.jsをインストールしてください


【エラー】
┌────────────────────────────────────┐
│ Error: listen EADDRINUSE          │
│ :::3000                           │
└────────────────────────────────────┘
【原因】ポート3000が既に使用されている
【対処】Ctrl+C でサーバーを止めてから再起動


【エラー】
┌────────────────────────────────────┐
│ npm ERR! EBADF: bad file          │
│ descriptor, write                 │
└────────────────────────────────────┘
【原因】Googleドライブ上で実行している
【対処】ローカルディスク（C:\）にコピー
