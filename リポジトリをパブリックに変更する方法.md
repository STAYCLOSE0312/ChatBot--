# 🔓 GitHubリポジトリをパブリックに変更する方法

## ⚠️ 重要

Renderなどの無料ホスティングサービスでは、**パブリックリポジトリ**が必要です。
現在のリポジトリは「Private」（プライベート）になっているため、Renderで接続できません。

## 🔧 変更手順

### ステップ1: GitHubのリポジトリページを開く

1. https://github.com/STAYCLOSE0312/ChatBot-- にアクセス
2. ログインしていることを確認

### ステップ2: 設定を開く

1. リポジトリページの上部にあるタブから「**Settings**」（設定）をクリック
2. 左側のメニューを下にスクロール
3. 「**Danger Zone**」（危険ゾーン）セクションを探す

### ステップ3: 可視性を変更

1. 「**Change visibility**」（可視性を変更）をクリック
2. 「**Change to public**」（パブリックに変更）をクリック
3. 確認ダイアログが表示されます：
   - リポジトリ名を入力して確認（例: `STAYCLOSE0312/ChatBot--`）
   - 「**I understand, change repository visibility**」をクリック

### ステップ4: 確認

1. リポジトリページに戻る
2. リポジトリ名の横に「**Public**」と表示されていれば成功です

## ✅ 完了

これで、Renderでリポジトリを接続できるようになりました！

## 🔐 セキュリティについて

**安心してください**：
- `.env`ファイルと`credentials`フォルダは`.gitignore`で除外されているため、GitHubにアップロードされていません
- 認証情報が公開されることはありません
- コードのみが公開されます

## 📝 次のステップ

リポジトリをパブリックに変更したら、次はRenderで公開します：

1. https://render.com にアクセス
2. 「Get Started for Free」→「Continue with GitHub」
3. 「New +」→「Web Service」
4. `ChatBot--`リポジトリを選択
5. 設定を入力してデプロイ

詳細は `公開手順_超簡単版.md` を参照してください。
