# 🔍 GitHub連携の確認方法

## 📋 確認方法1: Cursorのソース管理パネルで確認

### ステップ1: ソース管理パネルを開く

1. Cursorの左側のサイドバーで「ソース管理」アイコンをクリック
   - または `Ctrl + Shift + G`

### ステップ2: 連携状態を確認

以下のいずれかが表示されていれば、GitHubと連携されています：

#### ✅ 連携済みの場合

- 上部に「変更を同期」または「プッシュ」ボタンが表示される
- または、「ブランチ名: main」と表示される
- または、リモートリポジトリのURLが表示される

#### ❌ 未連携の場合

- 「ブランチを発行」または「GitHubに発行」ボタンが表示される
- または、リモートリポジトリの情報が表示されない

---

## 📋 確認方法2: Cursorのターミナルで確認

### ステップ1: ターミナルを開く

1. Cursorの下部の「ターミナル」タブをクリック
   - または `Ctrl + `` `（バッククォート）

### ステップ2: コマンドを実行

以下のコマンドを入力してEnterキーを押します：

```bash
git remote -v
```

#### ✅ 連携済みの場合

以下のような表示が出ます：

```
origin  https://github.com/YOUR_USERNAME/faq-chatbot.git (fetch)
origin  https://github.com/YOUR_USERNAME/faq-chatbot.git (push)
```

#### ❌ 未連携の場合

何も表示されないか、エラーメッセージが表示されます。

---

## 📋 確認方法3: GitHubのWebサイトで確認

### ステップ1: GitHubにアクセス

1. https://github.com にアクセス
2. ログイン

### ステップ2: リポジトリを確認

1. 右上のアイコンをクリック
2. 「Your repositories」をクリック
3. `faq-chatbot`（または作成したリポジトリ名）が表示されていれば成功

---

## 🔧 未連携の場合の対処法

### 方法1: CursorのUIから連携（推奨）

1. **ソース管理パネル**を開く（`Ctrl + Shift + G`）
2. 「変更」の下にファイルがあることを確認
3. 「すべてステージ」ボタンをクリック
4. メッセージ欄に「初回アップロード」と入力
5. 「✓」または「コミット」をクリック
6. 「ブランチを発行」ボタンをクリック
7. 「パブリックリポジトリとしてGitHubに発行」を選択
8. リポジトリ名を入力（例: `faq-chatbot`）
9. 「GitHubに発行」をクリック

### 方法2: ターミナルから連携

1. **ターミナル**を開く（`Ctrl + `` `）
2. 以下のコマンドを順番に実行：

```bash
# ステップ1: Gitを初期化（まだの場合）
git init

# ステップ2: ファイルを追加
git add .

# ステップ3: コミット
git commit -m "初回アップロード"

# ステップ4: GitHubリポジトリを接続
# ⚠️ YOUR_USERNAME をあなたのGitHubユーザー名に置き換えてください！
git remote add origin https://github.com/YOUR_USERNAME/faq-chatbot.git

# ステップ5: ブランチ名を設定
git branch -M main

# ステップ6: GitHubにアップロード
git push -u origin main
```

## ✅ 連携確認チェックリスト

- [ ] Cursorのソース管理パネルに「変更を同期」ボタンが表示される
- [ ] `git remote -v` コマンドでリモートリポジトリのURLが表示される
- [ ] GitHubのWebサイトでリポジトリが表示される
- [ ] GitHubのリポジトリにファイルがアップロードされている

## 💡 補足情報

### GitHubユーザー名の確認方法

1. GitHubにログイン
2. 右上のアイコンをクリック
3. ユーザー名が表示されます

### リポジトリ名の確認方法

1. GitHubにログイン
2. 「Your repositories」をクリック
3. リポジトリ名が表示されます
