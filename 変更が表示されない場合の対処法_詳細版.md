# 🔧 変更が表示されない場合の対処法（詳細版）

## 📋 確認手順

### ステップ1: ファイルを保存

1. `lib/googleSheets.js` を開いた状態で
2. **`Ctrl + S`** を押して保存
3. ファイル名の横の「●」マークが消えれば保存完了

### ステップ2: ソース管理パネルをリフレッシュ

1. **`Ctrl + Shift + G`** でソース管理パネルを開く
2. または、ソース管理パネルを一度閉じて再度開く
3. または、`F5`キーでリフレッシュ

### ステップ3: Gitの状態を確認

ターミナルで以下を実行：

```bash
git status
```

変更されたファイルが表示されるはずです。

---

## 🔄 それでも表示されない場合

### 方法1: ファイルを一度閉じて再度開く

1. `lib/googleSheets.js` のタブを閉じる
2. エクスプローラーから再度開く
3. ソース管理パネルを確認

### 方法2: Cursorを再起動

1. Cursorを完全に閉じる
2. 再度開く
3. ソース管理パネルを確認

### 方法3: 手動でGitコマンドを実行

ターミナルで以下を実行：

```bash
# 変更を確認
git status

# ファイルを追加
git add lib/googleSheets.js

# 状態を確認
git status
```

これで変更が検出されるはずです。

---

## ✅ 変更が表示されたら

1. 「すべてステージ」をクリック
2. メッセージ欄に「認証情報の読み込み処理を改善」と入力
3. 「✓」をクリック
4. 「変更を同期」または「プッシュ」をクリック
